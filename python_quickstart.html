Szybkie wprowadzenie¶

Zacznijmy od uruchomienia Pythona:

~$ python
Python 3.7.0 (...)
Type "copyright", "credits" or "license" for more information.

>>>

Znak zachęty >>> (otworzyliśmy tryb interaktywny Pythona, nie przepisujemy znaku zachęty!)

Przetestujmy Pythona: 2 + 2:

>>> 2 + 2
4

Python jako kalkulator:

>>> 6 * 7
42
>>> 1252 - 38 * 6
1024
>>> (3 - 5) * 7
-14
>>> 21 / 7
3.0
>>> 3**2
9
>>> 5 / 2
2.5

Zapis liczb dziesiętnych: używamy kropki, nie przecinka. Przecinki w Pythonie są używane do np. definiowania krotek (ale o tym będzie później).

Operatory arytmeryczne:

  • dodawnie +

  • odejmowanie -

  • mnożenie *

  • dzielenie /

  • dzielenie modulo %

  • dzielenie “podłoga” //

    >>> 7 // 3
2
>>> 7 % 2
1

Przedstaw się!¶

String¶

Typ strings - czyli łańcuchy znaków = napisy

Przykłady:

>>> 'Hello World'
'Hello World'
>>> "Ala ma kota"
'Ala ma kota'
>>> "Rock 'n' Roll"
"Rock 'n' Roll"
>>> 'Rock \'n\' Roll'
"Rock 'n' Roll"
>>> 'Najlepszą książką kulinarną roku została "Jadłonomia"'
Najlepszą książką kulinarną roku została "Jadłonomia"'

Ciągi znaków można dodawać:

>>> 'Najlepszy film w historii kina to' + '"XXX"'
'Najlepszy film w historii kina to "XXX"'

lub mnożyć za pomocą liczb całkowitych:

>>> 'Hello' * 3
'HelloHelloHello'

czy nazwać wartość, by ją ponownie użyć:

>>> title = '"Obywatel Kane"'
>>> 'Najlepszy film w historii kina to ' + title
'Najlepszy film w historii kina to "Obywatel Kane"'
>>> 'W weekend leci ' + title + ' w telewizji'
'W weekend leci "Obywatel Kane" w telewizji'

Ciągi znaków zawsze zaczynają się tym samym znakiem. Za pomocą apostrofu (') lub cudzysłowu ("). Nie ma to wpływu na wyświetlanie stringa np. "Abrakadabra" wyświetli napis Abrakadabra - cudzysłów nie jest częścią napisu, wskazuje, że będziemy podawać łańcuch znaków czyli napis.

Wyświetlanie napisów¶

Używamy funkcji wbudowanej print() :

>>> print("Hello World")
Hello World

Wyświetlanie kilku napisów bez dodawania ich do siebie. Będą oddzielone spacją:

>>> print("Hi, my name is", "Rita")
Hi, my name is Rita

Inne przykłady:

>>> print(1)
1
>>> print(1, 2, 3)
1 2 3
>>> print("2 + 2 =", 2 + 2)
2 + 2 = 4

>>> quit()

Możemy skorzystać ze skrótu klawiszowego Ctrl+D (Mac/Linux) lub Ctrl+Z (Windows).

Pliki źródłowe¶

W trybie interaktywnym wydajemy polecenia osobno i natychmiast otrzymujemy odpowiedź.

Znacznie wygodniej trzymać kod w pliku. Kod edytujemy w PyCharmie.

Nasz pierwszy program wyglądał następująco:

print("Hello World")

Stwórzmy nowy plik triangle.py.

~$ python triangle.py
+
++
+++
~$

Pojedynczy program może zawierać więcej niż jedno polecenie. Na przykład:

print("Hello world")
print()

print("my name is Rita")

print()
print("Bye.")

Możemy wstawić puste linie w pliku hello.py, żeby poprawić czytelność.

BMI kalculator¶

Najsłynniejsze zadanie z podstaw programowania. Kalkulator BMI (Body Mass Index). Wzór jest bardzo prosty:

BMI = (masa (kg)) / (wzrost (m))²

Stwórz plik bmi.py i napisz program, który obliczy BMI:

Your BMI is: 20.56880661366243

print("Your BMI is:", 52 / (1.59 ** 2))

Notatki po zajęciach¶

Rzutowanie¶

Kiedy mówimy o wartościach liczbowych w najprostrzym przypadku mamy na myśli liczby całkowite (ang. integer) oraz dziesiętne, zwane też zmiennoprzecinkowe (ang. floating point).

Zamiana jednego typu na drugi jest dla nas intuicyjna. Liczba całkowita 5 to zmiennoprzecinkowo 5.0. A najbliższą liczbą całkowitą od 5.4 jest po prostu 5.

Dla komputera informację tę musimy przedstawić przez rzutowanie (inaczej konwersja), czyli zmianę jednego typu na drugi.

>>> a = 7
>>> b = 7.6

>>> a_to_float = float(a)
7.0
>>> b_to_int = int(b)
7

Łatwo zauważyć, że zamiana liczby dziesiętnej na całkowitą nie działa jak zaokrąglenie, ale powododuje utratę informacji przez obcięcie wartości po przecinku.

Innym przykładem może być zmiana tekstu na liczbę:

>>> txt = '124'
>>> txt + 4
TypeError: can only concatenate str (not "int") to str
>>> num = int(txt)
>>> num + 4
128

>>> txt = '124.345'
>>> txt + 4.3
TypeError: can only concatenate str (not "float") to str
>>> num = float(txt)
>>> num + 4.3
124.645

Oraz odwrotnie zmiana liczby na tekst:

>>> num = 6
>>> 'Ala ma ' + num + ' kotów'
TypeError: can only concatenate str (not "int") to str
>>> txt = str(num)
>>> 'Ala ma ' + txt + ' kotów'
'Ala ma 6 kotów'
